# Naive Bayer Classifier for text classification

This code implements in Java a simple naive bayes classfier for documents.

## Getting Started

### Prerequisites

* JDK 6+
* EJML 0.19+
* GNUPlot 4.2+ (for use graphics) 
* JavaPlot (for use graphics) 
* Weka 3.6.7+
